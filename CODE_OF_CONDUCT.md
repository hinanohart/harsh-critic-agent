# Code of Conduct

This project adopts the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
version 2.1.

## Summary

- Be respectful. Criticize ideas, not people.
- Harsh feedback on *work* is this project's entire point; harsh
  feedback on *people* is not welcome.
- Assume good faith.
- Report unacceptable behaviour privately to `hinanohart@gmail.com`.

## Scope

Applies in all project spaces — issues, pull requests, discussions, and
any other channel where participants represent the project.

## Enforcement

Reports are reviewed promptly and handled with discretion. Consequences
range from a private warning to a permanent ban depending on severity.
